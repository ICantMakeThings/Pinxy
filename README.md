# Pinxy
Pixie v4 Mod

Outputs:


0.6W on 40/80M

0.2W on 20/30M


Step sizes: 1kHz, 10kHz, 100kHz

Twist to go up/down, press encoder to change step size, hold encoder for about 1 second to change band.
